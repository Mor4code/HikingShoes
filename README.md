# HikingShoes
A one-page website for a fictional store

## Instructions on how to run the project
-I removed the node module, so its important to install.

1.Clone the repository:
git clone <repository-url>
cd HikingShoes

2.Install dependencies:
npm install

3.Start the development server:
npm run dev

4.Optional) Build for production:
npm run build

5.(Optional) Start the production server:
npm run start

## Any challenges you faced and how you overcame them
At first, I struggled with installing libraries, but once I understood the process, I was able to implement components successfully. It was also my first experience using Tailwind, so I had to learn the basics and understand how standard CSS translates into Tailwind classes.
